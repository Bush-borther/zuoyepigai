## English

Thank you for your interest in contributing to the Smart Exam Grading System!

### How to Contribute

1. **Fork the Repository**
2. **Create a Feature Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make Your Changes**
4. **Test Your Changes**
5. **Commit with Clear Messages**
   ```bash
   git commit -m "Add: description of your changes"
   ```
6. **Push to Your Fork**
   ```bash
   git push origin feature/your-feature-name
   ```
7. **Create a Pull Request**

### Code Style

- **Python**: Follow PEP 8
- **TypeScript/React**: Use ESLint and Prettier
- **Comments**: Add bilingual comments (English and Chinese) for important logic
